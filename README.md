# AI Maze Solver

This project implements an Artificial Intelligence based Maze Solver.

Features:
- Maze representation using Grid
- BFS (Breadth First Search)
- DFS (Depth First Search)
- Uniform Cost Search (UCS)
- A* Search using Manhattan Distance Heuristic
- Constraint Satisfaction using Backtracking
- Decision Making Agent using Utility Function
- Reasoning Under Uncertainty
- Maze Visualization GUI

Technologies Used:
- Python
- HTML
- CSS
- JavaScript

How to Run:

python main.py
